# Organization profile for tamer4lynx

This folder contains the content for the [tamer4lynx](https://github.com/tamer4lynx) organization profile on GitHub.

## Setup

1. On GitHub, create a new **public** repository named `.github` in the tamer4lynx organization (or use "Create repository" from the org profile prompt).
2. Add the contents of `profile/` to the repo:
   - `profile/README.md` → `.github/profile/README.md`
   - `profile/tamer-logo.png` → `.github/profile/tamer-logo.png`
3. Commit and push. The README will appear on https://github.com/tamer4lynx.
